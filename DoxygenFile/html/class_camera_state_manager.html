詳解

Dxlibの3Dのカメラの処理を行うクラス

カメラを，上から見るのか，横から見るのか，切り替えるのはDXlibの場合操作が少々難しい
このクラスはその処理をまとめたことで，処理を簡単にしている
また，カメラの姿勢はクォータニオンで表している．
