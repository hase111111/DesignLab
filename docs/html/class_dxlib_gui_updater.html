DxlibGuiUpdater クラスfinal

クリック判定を行うクラス．
一度のクリックで複数のGUIが反応することを防ぐために，優先度を設定する．
優先度が高いものから順にクリック判定を行い，クリックされたらそのGUIのActivate関数を実行する．
神クラスになっている感は否めないが，また問題が起きたら修正する． [詳解]

詳解

クリック判定を行うクラス．
一度のクリックで複数のGUIが反応することを防ぐために，優先度を設定する．
優先度が高いものから順にクリック判定を行い，クリックされたらそのGUIのActivate関数を実行する．
神クラスになっている感は否めないが，また問題が起きたら修正する．
