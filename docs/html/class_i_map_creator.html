◆ InitMap()

virtual MapState IMapCreator::InitMap ( )
pure virtual

マップの初期化を行う．

シミュレーションにおいてはこの機能のみあればよいが，
実機を動作させる場合，カメラが正確に認識できる距離の関係で，マップを読み直す必要がある．
そのため，実機試験時はこちらの関数で初期化した後，UpdateMap()でマップを更新する必要がある．

戻り値
初期化したマップ

MapCreatorByCsv, MapCreatorForSimulationで実装されています。
