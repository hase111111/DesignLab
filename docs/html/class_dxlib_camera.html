詳解

Dxlibの3Dのカメラの状態を管理するクラス．

カメラを，上から見るのか，横から見るのか，切り替えるのはDXlibの場合操作が少々難しい．
また，Dxlibは関数で処理を書くため，現在のカメラの状態を保持してまとめておく処理はデフォルトにない．
このクラスはその処理をまとめたことで，処理を簡単にしている
また，カメラの姿勢はクォータニオンで表している．
デフォルトは kDefaultCameraFrontVecの方を向いて，kDefaultCameraUpVecがカメラの上方向を表す．
