詳解

PhantomXのパラメータを定数で表現したもの．
コンストラクタを削除したので，実体は生成できない．( PhantomXConst::kLegNum みたいに値を呼び出すこと )

簡単のため値をここにまとめたが，むやみにこの値を参照せずにHexapodStateCalculatorを使うこと．
設計的にあまり正しくない気がするが，現状効率よく開発できているので問題が起きたときに考えることにする．
また，座標系はロボット前方にx軸，左方向にy軸，上方向にz軸をとる右手座標系である．
